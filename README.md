## 프로젝트 소개
### 프로젝트 개요
사용자들이 더 간편하게 합리적인 건강검진을 예약하고 건강 상태를 체크할 수 있는 모바일 어플리케이션

### 주요 기능
    * 제휴 병원 목록 제공 (전체, 지역별) 
    * 병원 위치 제공
    * 건강검진 예약 서비스 
    * 유저들의 후기를 통한 병원 비교  
    * 병원 관계자와의 의료 상담 
    * 유저가 실시하는 건강 테스트



## 시작 가이드
**Requirements** 

For building and runnig the application you need:
* Node.js v18.19.1
* Npm 9.2.0
* yarn 1.22.21  
  
**Installation**
```
$ git clone https://github.com/HJNYJ/mediDoc-v2.0.git
$ cd mediDoc-v2.0
$ yarn
```

**Run**
```bash
yarn dev
# or
yarn build
yarn start
```
